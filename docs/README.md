## 🚀 快速開始

### 新開發者

1. 閱讀 [README.md](../README.md) 了解專案概況
2. 閱讀 [開發指南](./開發指南.md) 設定開發環境
3. 閱讀 [專案結構與路徑說明](./專案結構與路徑說明.md) 了解專案結構

### 新增題目資料

1. 閱讀 [開發指南 - 新增題目資料](./開發指南.md#1-新增題目資料)
2. 執行路徑驗證：`node scripts/validatePaths.js`

### 修改配置

1. 閱讀 [題庫配置使用指南](./題庫配置使用指南.md)
2. 修改 `assets/config/quiz-library-config.json`
3. 重新啟動應用程式

## 📖 文檔維護

### 更新文檔

當專案結構或功能變更時，請記得更新相關文檔：

- **新增功能** → 更新 [開發指南](./開發指南.md)
- **路徑變更** → 更新 [專案結構與路徑說明](./專案結構與路徑說明.md)
- **腳本變更** → 更新 [檔案功能說明](./檔案功能說明.md)
- **配置變更** → 更新 [題庫配置使用指南](./題庫配置使用指南.md)

### 文檔格式

- 使用 Markdown 格式
- 使用繁體中文
- 包含清晰的標題和目錄
- 提供程式碼範例和命令
