### 3. 題目內容圖片

#### 3.1 題目圖片（動態載入）
- **檔案路徑**：`assets/images/{testName}_{subject}_{series_no}/`
- **檔案格式**：PNG
- **命名規則**：
  - 題目圖片：`{題號}Q{序號}.png`（例如：`41Q1.png`, `42Q1.png`）
  - 選項圖片：`{題號}{選項}{序號}.png`（例如：`45A1.png`, `45B1.png`, `45C1.png`, `45D1.png`）
- **功能**：題目內容中嵌入的圖片
- **載入方式**：
  - Web 平台：使用相對路徑 `/assets/images/...`
  - 原生平台：使用 URI 方式，Expo 自動處理
- **圖片大小範例**（以 `IPAS_02_L23_11411` 為例）：
  - `41Q1.png`: 45,271 bytes (約 45 KB)
  - `42Q1.png`: 14,669 bytes (約 15 KB)
  - `43Q1.png`: 14,669 bytes (約 15 KB)
  - `44Q1.png`: 14,669 bytes (約 15 KB)
  - `45Q1.png`: 14,669 bytes (約 15 KB)
  - `45A1.png`: 22,528 bytes (約 23 KB)
  - `45B1.png`: 24,441 bytes (約 24 KB)
  - `45C1.png`: 24,629 bytes (約 25 KB)
  - `45D1.png`: 19,849 bytes (約 20 KB)

#### 3.2 圖片使用方式
- **在題目內容中**：使用 `@@` 標記符號標記圖片位置
- **圖片路徑推斷**：系統會根據以下資訊自動構建圖片路徑：
  - `testName`（測驗名稱，如：`IPAS_02`）
  - `subject`（科目，如：`L23`）
  - `series_no`（期數，如：`11411`）
  - `questionNumber`（題號，如：`45`）
  - `optionLabel`（選項標籤，如：`A`, `B`, `C`, `D`）
- **完整路徑範例**：`assets/images/IPAS_02_L23_11411/45A1.png`

## 🔧 圖片載入機制

### Web 平台
- 使用相對路徑：`/assets/images/{路徑}`
- 圖片會被打包到 Web 版本的 `assets` 目錄

### 原生平台（iOS/Android）
- 使用 URI 方式：`assets/images/{路徑}`
- Expo 會自動處理 `assets` 目錄下的資源
- 圖片會被打包到應用程式中

### 圖片載入組件
- **組件**：`RichTextWithImages`
- **功能**：
  - 解析題目文字中的 `@@` 標記
  - 自動構建圖片路徑
  - 顯示載入中的狀態
  - 支援點擊圖片查詢功能

## ⚙️ 配置說明

### app.json 配置
```json
{
  "assetBundlePatterns": [
    "**/*",
    "assets/data/**/*.json",
    "assets/data/questions/**/*.json",
    "assets/images/**/*.png",
    "assets/images/**/*.jpg",
    "assets/images/**/*.jpeg"
  ]
}
```

這確保所有圖片資源都會被打包到應用程式中。

## 📝 注意事項

1. **圖片優化**：建議使用圖片壓縮工具優化圖片大小，減少應用程式體積
2. **命名規範**：嚴格遵守命名規則，確保系統能正確載入圖片
3. **路徑一致性**：確保圖片路徑與 JSON 資料中的標記一致
4. **檔案格式**：優先使用 PNG（支援透明背景），大圖可使用 JPG（檔案較小）
5. **解析度**：提供適當解析度的圖片，避免過大影響載入速度
